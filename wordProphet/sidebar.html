Instructions

This Shiny App predicts the next word based on the text you type.

The input and output fields are as follows.

  • Enter a word or phrase: Text entered for prediction.
  • Number of Predictions: Number of words that the algorithm will predict.
  • Input Text: The input text.
  • Predicted Words: The predicted next word(s), shown when the app detects that you have finished typing one or more words.

    • Please allow a few seconds for the output to appear.
